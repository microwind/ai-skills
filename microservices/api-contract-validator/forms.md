## 使用说明

1. **配置顺序**: 按照验证器基础配置 → 契约文件配置 → 验证规则配置 → 验证执行配置 → 错误处理配置 → 集成配置 → 监控配置 → 报告配置 → 高级配置 → 安全配置 → 测试配置的顺序进行
2. **基础设置**: 配置验证器类型、名称、描述和API规范版本
3. **文件设置**: 配置契约文件来源、文件路径、文件解析和预处理
4. **规则设置**: 配置基础验证、高级验证、业务规则和安全验证
5. **执行设置**: 配置执行模式、验证范围、并行处理和资源限制
6. **错误设置**: 配置错误级别、处理策略、错误报告和报告输出
7. **集成设置**: 配置CI/CD集成、IDE集成、质量门禁和代码辅助
8. **监控设置**: 配置性能监控、监控方式、告警规则和告警通知
9. **报告设置**: 配置报告内容、报告格式和报告生成
10. **高级设置**: 配置自定义验证器、插件管理、扩展功能和版本管理
11. **安全设置**: 配置访问控制、安全策略和审计配置
12. **测试设置**: 配置测试用例、测试配置、测试执行和结果验证
13. **启动验证**: 启动API契约验证器并验证配置
