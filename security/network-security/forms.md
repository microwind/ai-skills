## 使用说明

1. **配置顺序**: 按照项目配置 → 网络扫描 → Web安全 → 系统安全的顺序进行
2. **扫描策略**: 根据目标环境选择合适的扫描策略和参数
3. **安全考虑**: 确保扫描活动不会对生产环境造成影响
4. **合规要求**: 根据行业和地区要求配置相应的合规检查
5. **报告生成**: 配置适当的报告格式和详细程度
6. **持续监控**: 建立持续的安全监控和告警机制
