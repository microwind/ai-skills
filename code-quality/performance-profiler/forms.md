## 使用说明

1. **配置顺序**: 按照性能分析基础配置 → 分析范围配置 → CPU性能分析配置 → 内存性能分析配置 → I/O性能分析配置 → 分析执行配置 → 数据收集配置 → 报告生成配置 → 高级配置的顺序进行
2. **基础设置**: 配置项目名称、分析类型、分析目标、编程语言和分析模式
3. **范围设置**: 配置源代码扫描范围、函数选择和过滤条件
4. **CPU分析**: 配置CPU分析工具、分析策略、线程分析和并发分析
5. **内存分析**: 配置内存分析工具、内存指标、内存泄漏检测和对象分析
6. **I/O分析**: 配置文件I/O分析、网络I/O分析和相关指标
7. **执行配置**: 配置运行环境、执行参数、资源限制和执行策略
8. **数据收集**: 配置数据采集、存储和清理策略
9. **报告设置**: 配置报告格式、内容、结构和可视化选项
10. **高级功能**: 配置实时分析、分布式分析和AI辅助分析
11. **执行分析**: 启动性能分析过程并监控执行状态
12. **结果验证**: 验证分析结果的准确性和完整性
