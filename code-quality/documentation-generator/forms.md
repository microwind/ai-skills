## 使用说明

1. **配置顺序**: 按照文档生成基础配置 → 源代码分析配置 → 文档结构配置 → 模板配置 → 生成配置 → 质量控制配置 → 多语言配置 → 高级配置的顺序进行
2. **基础设置**: 配置项目名称、文档类型、目标语言、编程语言和输出格式
3. **代码分析**: 配置源代码扫描范围、元素提取和类型信息
4. **文档结构**: 配置目录结构、页面组织和内容策略
5. **模板设置**: 配置文档模板、样式和自定义选项
6. **生成配置**: 配置输出设置、生成选项和缓存策略
7. **质量控制**: 配置内容验证、质量指标和修复建议
8. **多语言设置**: 配置支持语言、翻译策略和本地化
9. **高级功能**: 配置插件扩展、集成和部署选项
10. **执行生成**: 启动文档生成过程并监控生成状态
11. **结果验证**: 验证生成的文档质量和完整性
