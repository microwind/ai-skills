## 使用说明

1. **配置顺序**: 按照重构项目基础配置 → 代码分析配置 → 重构模式选择 → 高级重构模式 → 重构执行配置 → 重构效果评估的顺序进行
2. **基础设置**: 配置项目名称、重构目标、重构范围、编程语言和重构模式
3. **代码分析**: 配置代码质量分析工具、分析指标和代码异味检测
4. **模式选择**: 根据分析结果选择合适的重构模式并配置重构参数
5. **高级模式**: 配置设计模式引入、架构重构等高级重构策略
6. **执行配置**: 配置重构执行策略、测试策略和风险控制措施
7. **效果评估**: 配置质量指标评估、业务价值评估和效果验证方法
8. **执行重构**: 按照配置执行重构过程并监控执行状态
9. **结果验证**: 验证重构效果和业务价值实现
