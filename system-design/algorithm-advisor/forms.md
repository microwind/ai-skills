# 算法顾问参数配置表单

## 核心参数配置

### 问题类型
- **数据规模**: 
  - [ ] 小规模 (< 1K)
  - [ ] 中规模 (1K - 1M) 
  - [ ] 大规模 (1M - 100M)
  - [ ] 超大规模 (> 100M)
- **数据类型**:
  - [ ] 数值型
  - [ ] 文本型
  - [ ] 图结构
  - [ ] 时间序列
  - [ ] 多媒体
- **操作类型**:
  - [ ] 查询/搜索
  - [ ] 排序
  - [ ] 聚类/分类
  - [ ] 路径规划
  - [ ] 推荐系统

### 性能约束
- **时间复杂度要求**:
  - [ ] O(1) - 常数时间
  - [ ] O(log n) - 对数时间
  - [ ] O(n) - 线性时间
  - [ ] O(n log n) - 线性对数时间
  - [ ] O(n²) - 平方时间
- **空间复杂度要求**:
  - [ ] O(1) - 常数空间
  - [ ] O(log n) - 对数空间
  - [ ] O(n) - 线性空间
  - [ ] O(n²) - 平方空间
- **响应时间要求**:
  - [ ] < 10ms (实时)
  - [ ] < 100ms (近实时)
  - [ ] < 1s (交互式)
  - [ ] < 10s (批处理)

### 系统约束
- **内存限制**:
  - [ ] < 1GB
  - [ ] 1-8GB
  - [ ] 8-64GB
  - [ ] > 64GB
- **CPU核心数**:
  - [ ] 单核
  - [ ] 少核 (2-8)
  - [ ] 多核 (8-32)
  - [ ] 众核 (> 32)
- **分布式要求**:
  - [ ] 单机
  - [ ] 多机
  - [ ] 跨数据中心
  - [ ] 边缘计算

### 业务约束
- **准确性要求**:
  - [ ] 精确结果
  - [ ] 近似结果
  - [ ] 概率结果
- **一致性要求**:
  - [ ] 强一致性
  - [ ] 最终一致性
  - [ ] 弱一致性
- **可用性要求**:
  - [ ] 99.9% (三个九)
  - [ ] 99.99% (四个九)
  - [ ] 99.999% (五个九)

## 校验规则

### 必填项检查
- [ ] 问题类型已选择
- [ ] 性能约束已定义
- [ ] 系统约束已明确

### 合理性检查
- [ ] 时间复杂度与数据规模匹配
- [ ] 空间复杂度与内存限制匹配
- [ ] 响应时间要求现实可行
- [ ] 分布式需求与业务场景匹配

### 完整性检查
- [ ] 所有约束条件已考虑
- [ ] 权衡取舍已明确
- [ ] 边界条件已定义
- [ ] 扩展性需求已考虑

## 使用说明

1. **填写顺序**: 按照问题类型 → 性能约束 → 系统约束 → 业务约束的顺序填写
2. **优先级**: 如果约束冲突，优先满足业务约束，然后是性能约束
3. **权衡**: 对于相互矛盾的约束，明确优先级和可接受的折中方案
4. **验证**: 填写完成后运行校验规则，确保配置的合理性
