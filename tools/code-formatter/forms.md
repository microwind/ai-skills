## 使用说明

1. **配置顺序**: 按照项目配置 → 语言配置 → 文件目录配置 → 格式化规则 → 验证检查 → 输出配置的顺序进行
2. **语言选择**: 根据项目类型选择合适的格式化工具和配置
3. **文件选择**: 正确设置目标文件和目录，配置过滤规则
4. **规则设置**: 根据团队标准配置格式化规则和验证规则
5. **验证检查**: 启用适当的验证和质量检查
6. **输出设置**: 配置输出格式和报告生成
7. **集成配置**: 根据需要配置IDE、CI/CD和Git集成
