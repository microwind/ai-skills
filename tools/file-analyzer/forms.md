## 使用说明

1. **配置顺序**: 按照项目配置 → 文件选择 → 分析配置 → 输出配置的顺序进行
2. **文件选择**: 确保文件路径正确，支持多种过滤条件
3. **分析配置**: 根据文件类型选择合适的分析选项
4. **性能优化**: 合理配置并发和缓存参数
5. **输出设置**: 选择合适的报告格式和可视化选项
6. **集成配置**: 根据需要配置IDE和CI/CD集成
