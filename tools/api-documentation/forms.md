## 使用说明

1. **配置顺序**: 按照项目配置 → API源配置 → 文档结构配置 → 数据模型配置 → 请求响应配置 → 生成配置的顺序进行
2. **API源选择**: 确保API源路径正确，支持多种源类型
3. **文档结构**: 合理配置文档结构和分组，提高可读性
4. **数据模型**: 完整定义数据模型和验证规则
5. **安全配置**: 根据实际需求配置认证和权限
6. **生成设置**: 选择合适的输出格式和模板
7. **集成配置**: 根据需要配置CI/CD和监控集成
