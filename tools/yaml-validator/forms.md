## 使用说明

1. **配置顺序**: 按照基础验证配置 → YAML语法配置 → 数据类型验证 → 结构验证配置 → 安全验证配置 → 性能配置 → 输出配置 → 集成配置 → 高级功能的顺序进行
2. **安全优先**: 始终优先配置安全验证选项，特别是处理外部来源的YAML文件
3. **性能优化**: 根据文件大小和数量调整性能配置参数
4. **集成设置**: 根据开发环境配置相应的集成选项
5. **测试验证**: 在正式使用前进行充分的测试验证
