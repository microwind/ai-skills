## 🚀 使用指南

### 快速开始
```bash
# 1. 克隆项目
git clone https://github.com/yourusername/ai-skills.git
cd ai-skills

# 2. 查看主README
cat README.md

# 3. 选择学习路径
# - 初级开发者 → SKILLS_BY_DIFFICULTY.md
# - 需要指导 → SKILLS_ROADMAP.md
# - 查找具体Skill → SKILLS_INDEX.md

# 4. 开始学习
cd backend/restful-api-design/
cat 中文说明.md
```

### 学习方式
1. **阅读** - 理解核心概念（中文说明）
2. **实践** - 运行示例代码（scripts目录）
3. **应用** - 在项目中使用
4. **反思** - 总结经验
