# 常见问题 (FAQ)

## 关于中文AI Skills库

### Q: 这个库是用来做什么的？
**A:** 这是一个为中文开发者提供的编程知识库，包含50+个精选的Skills，每个Skill都包含：
- 中文详细说明
- 核心概念讲解
- 实战代码示例
- 最佳实践
- 常见问题解答

### Q: Skills和教程有什么区别？
**A:**
- **教程** - 从头到尾的学习路径
- **Skill** - 针对特定问题的解决方案库

每个Skill都是自成一体的，可以随意查阅。

### Q: 这个库更新频率如何？
**A:**
- 核心Skills定期更新
- 跟进最新技术发展
- 欢迎贡献新内容

## 贡献与参与

### Q: 我能贡献新的Skills吗？
**A:** 当然可以！请遵循：
1. Fork本项目
2. 创建新分支（feature/skill-name）
3. 按照模板创建Skill
4. 提交PR，描述你的贡献

### Q: Skill模板是什么样的？
**A:** 每个Skill应包含：
```
skill-name/
├── SKILL.md           # 英文标准文档
├── 中文说明.md        # 中文详细说明
├── README.md          # 概览
├── scripts/           # 代码示例
├── references/        # 参考资源
└── assets/            # 模板等
```

### Q: 我发现了错误或不准确的地方，怎么反馈？
**A:**
- 提交Issue描述问题
- 提交PR修正错误
- 在讨论区讨论改进意见

### Q: 贡献新Skill有什么要求吗？
**A:**
- ✅ 内容准确可靠
- ✅ 包含实战代码示例
- ✅ 遵循现有格式
- ✅ 说明清晰易懂
- ✅ 遵循开源许可证

## 常见困惑

### Q: 我学了很多Skills，但还是不会做项目，为什么？
**A:** 这很正常！Skills是零散的知识块，需要：
1. **整合能力** - 把多个Skills组合使用
2. **架构设计** - 整体规划项目结构
3. **实战经验** - 在真实项目中应用
4. **反复迭代** - 通过做项目学习

**建议：** 从小项目开始，逐步做大型项目

### Q: 学习Skills的最好方式是什么？
**A:**
1. **被动学习** - 阅读文档和教程（30%）
2. **主动学习** - 动手写代码（50%）
3. **反思总结** - 写笔记和博客（20%）

重点是主动学习和实践！

### Q: 如何判断自己是否真的掌握了一个Skill？
**A:** 能够：
- [ ] 解释核心概念
- [ ] 写出实现代码
- [ ] 在项目中应用
- [ ] 教别人如何使用
- [ ] 知道什么时候用，什么时候不用

### Q: Skills太多了，怎么选择学习？
**A:**
1. **明确目标** - 你想做什么？
2. **查看路线图** - 根据职位选择
3. **从简到难** - 先学入门级
4. **按需学习** - 项目需要什么就学什么
