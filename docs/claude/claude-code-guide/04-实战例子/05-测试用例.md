**学习目标**：
- 学习全面的测试策略
- 掌握测试用例设计
- 理解 TDD 工作流

购物车模块有很多功能，手动写测试需要 3-4 小时。Claude Code 可以 30 分钟完成。

## 第三步：填充完整的测试

### 提示词

```bash
$ claude "
现在实现这些测试用例，要求：
1. 每个测试都有清晰的注释说明在测试什么
2. 使用 expect() 做断言
3. 包括 AAA 模式（Arrange-Act-Assert）
4. 使用 Mock 处理外部依赖
5. 覆盖率目标 >95%

使用 Jest 的完整 API
"
```

### 完整测试代码示例

```typescript
describe('CartService', () => {
  let cart: CartService;

  beforeEach(() => {
    cart = new CartService();
  });

  describe('addItem', () => {
    test('应该添加商品到空购物车', () => {
      // Arrange
      const productId = 'PROD-1';
      const quantity = 1;

      // Act
      cart.addItem(productId, quantity);

      // Assert
      const items = cart.getItems();
      expect(items).toHaveLength(1);
      expect(items[0]).toEqual(
        expect.objectContaining({
          productId,
          quantity
        })
      );
    });

    test('应该增加已存在商品的数量', () => {
      cart.addItem('PROD-1', 1);
      cart.addItem('PROD-1', 2);

      const items = cart.getItems();
      expect(items).toHaveLength(1);
      expect(items[0].quantity).toBe(3);
    });

    test('应该抛出错误：数量无效', () => {
      expect(() => cart.addItem('PROD-1', -1))
        .toThrow('数量必须大于 0');
      expect(() => cart.addItem('PROD-1', 0))
        .toThrow('数量必须大于 0');
    });

    test('应该抛出错误：超过最大购买量', () => {
      cart.addItem('PROD-1', 999);
      expect(() => cart.addItem('PROD-1', 1))
        .toThrow('超过最大购买量');
    });
  });

  describe('calculateTotal', () => {
    test('应该计算多个商品的总价', () => {
      cart.addItem('PROD-1', 2); // $10 × 2 = $20
      cart.addItem('PROD-2', 1); // $20 × 1 = $20
      expect(cart.calculateTotal()).toBe(40);
    });

    test('应该正确应用折扣', () => {
      cart.addItem('PROD-1', 2); // $20
      cart.applyDiscount(0.1); // 10% 折扣
      expect(cart.calculateTotal()).toBe(18); // $20 × 0.9
    });

    test('空购物车应该返回 0', () => {
      expect(cart.calculateTotal()).toBe(0);
    });
  });

  describe('性能', () => {
    test('添加 1000 个商品应该在 100ms 内完成', () => {
      const start = performance.now();

      for (let i = 0; i < 1000; i++) {
        cart.addItem(`PROD-${i}`, 1);
      }

      const end = performance.now();
      expect(end - start).toBeLessThan(100);
    });
  });
});
```

## 测试用例设计原则

| 原则 | 说明 | 例子 |
|------|------|------|
| 正常用例 | 功能正常工作 | 添加商品成功 |
| 边界情况 | 边界值 | 0, 1, max, min |
| 错误情况 | 异常输入 | 负数、null、空值 |
| 性能测试 | 性能满足要求 | 1000+ 操作 |
| 集成测试 | 与其他模块协作 | 购物车 + 支付 |
