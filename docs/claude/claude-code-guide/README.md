## 💡 核心概念

### Claude Code 的三个层级

1. **对话式编程**：与 Claude 通过自然语言对话，逐步迭代
2. **工作流驱动**：将复杂项目分解为多步骤工作流
3. **上下文记忆**：Claude 记住整个项目背景，提供连贯的建议

### 最高效的使用模式

```
对话 → 验证 → 迭代 → 文档 → 测试
```

不是：一次性生成所有代码，而是通过对话不断优化。

## ❓ 常见问题

**Q：学完这个指南需要多长时间？**
A：根据你的基础和目标。初级开发者 2-3 小时快速上手，深入学习可能需要 1-2 天。

**Q：需要编程基础吗？**
A：需要基本的编程知识。建议你已经会至少一种编程语言。

**Q：代码示例可以复制使用吗？**
A：完全可以！所有代码都经过测试，可以直接复制到你的项目中。

**Q：项目实战能否直接运行？**
A：可以。每个项目都包含完整的配置和启动说明。
