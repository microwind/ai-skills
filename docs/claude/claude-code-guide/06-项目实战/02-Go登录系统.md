# 项目 2：Go 用户登录系统

**高性能的用户认证系统**：用 Go 和 Gin 框架实现。

**技术栈**：Go 1.21 + Gin + PostgreSQL + Docker

---

## 项目概览

完成后你将拥有：

- ✅ 高性能的 REST API（登录、注册、Token 刷新等）
- ✅ PostgreSQL 数据库集成
- ✅ JWT 认证系统
- ✅ Docker 容器化部署
- ✅ 单元和集成测试
- ✅ 完整的项目文档

## 常见问题

**Q：如何修改 JWT 配置？**
A：编辑 `internal/utils/jwt.go` 中的 `jwtSecret` 和过期时间。

**Q：怎样集成 Redis 缓存？**
A：在 `internal/database/` 中添加 Redis 客户端。

**Q：如何增加 API 速率限制？**
A：在 `middleware` 中添加速率限制中间件。

**Q：能部署到 Kubernetes 吗？**
A：可以，提供 Dockerfile 后加入 k8s manifest 即可。
