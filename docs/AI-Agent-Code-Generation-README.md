## 关键术语速查

| 术语 | 含义 | 在哪看 |
|------|------|------|
| **SKILL** | Specialized Knowledge and Intelligence Layer，算法思想+问题定义+约束+验证 | Part 1、Part 2第2步 |
| **Prompt** | 指导AI的具体指令，基于SKILL生成 | Part 2第3步、Part 3原则2 |
| **验证标准** | 代码是否满足SKILL要求的标准（单元测试+性能基准） | Part 2第5步、Part 3陷阱5 |
| **黄金法则** | SKILL设计的最佳实践（一个SKILL对应一个单位） | Part 3原则1 |
| **陷阱** | 常见的错误做法和后果 | Part 3 5大陷阱 |
