1. **配置顺序**: 按照迁移验证基础配置 → 数据库连接配置 → 迁移范围配置 → 结构验证配置 → 数据验证配置 → 性能验证配置 → 业务验证配置 → 报告配置 → 高级配置 → 监控配置 → 测试配置的顺序进行
2. **基础设置**: 配置验证类型、名称、描述和数据库类型
3. **连接设置**: 配置源数据库和目标数据库的连接信息
4. **范围设置**: 配置表选择、数据范围和过滤条件
5. **结构设置**: 配置表结构验证、数据类型映射和验证策略
6. **数据设置**: 配置数据完整性验证、数据一致性验证和验证方法
7. **性能设置**: 配置查询性能验证、索引性能验证和性能对比
8. **业务设置**: 配置业务规则验证、业务流程验证和规则执行
9. **报告设置**: 配置验证报告、通知方式和报告格式
10. **高级设置**: 配置自定义验证、验证规则引擎和脚本执行
11. **监控设置**: 配置验证监控、告警规则和告警通知
12. **测试设置**: 配置验证测试、回归测试和基线管理
13. **启动验证**: 启动数据库迁移验证并验证配置
