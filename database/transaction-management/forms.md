## 使用说明

1. **配置顺序**: 按照事务基础配置 → 数据库连接配置 → 事务属性配置 → 事务回滚配置 → 分布式事务配置 → 事务监控配置 → 事务日志配置 → 事务测试配置 → 高级配置 → 安全配置的顺序进行
2. **基础设置**: 配置事务管理器类型、名称、描述和数据库类型
3. **连接设置**: 配置数据库连接信息、认证信息和事务连接配置
4. **属性设置**: 配置基础事务属性、传播行为、隔离级别、超时配置和只读事务
