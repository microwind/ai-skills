# Cloud Native

## 概述

云原生技术栈，涵盖容器化、微服务、DevOps等现代化应用开发和部署方法。云原生是一种构建和运行应用程序的方法，充分利用云计算的弹性和分布式优势。

## 目标

- 掌握云原生技术栈的核心概念和实践
- 指导容器化和微服务架构的实施
- 提供CI/CD流水线和自动化部署方案
- 分享云原生应用的最佳实践和经验

## 适用场景

- 云原生应用开发
- 微服务架构部署
- 容器化项目实施
- DevOps流程构建
- 分布式系统迁移

## 主要内容

- **容器技术**: Docker容器化、镜像构建、容器编排
- **Kubernetes**: K8s集群管理、服务发现、负载均衡
- **微服务**: 服务拆分、服务网格、分布式系统设计
- **CI/CD**: 持续集成、持续部署、自动化流水线
- **监控运维**: 日志收集、性能监控、故障排查
- **云服务**: AWS、Azure、GCP等云平台使用
