## 使用说明

1. **配置顺序**: 按照集群信息 → 资源配置 → 网络配置 → 工作负载配置的顺序填写
2. **环境区分**: 不同环境使用不同的配置文件和命名空间
3. **安全优先**: 生产环境必须启用安全配置和RBAC
4. **监控配置**: 重要服务必须配置健康检查和监控告警
5. **备份策略**: 生产环境必须配置数据备份和集群备份
