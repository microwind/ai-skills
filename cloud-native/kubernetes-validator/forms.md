## 使用说明

1. **配置顺序**: 按照验证基础配置 → 集群配置 → 验证规则配置 → 验证标准配置 → 报告配置 → 告警配置 → 集成配置 → 高级配置的顺序进行
2. **基础设置**: 配置验证类型、名称、描述和模式
3. **集群设置**: 配置集群连接、集群范围和命名空间
4. **规则设置**: 配置资源验证、安全验证、存储验证和网络验证规则
5. **标准设置**: 配置合规标准、标准版本和自定义规则
6. **报告设置**: 配置报告格式、报告生成和报告分发
7. **告警设置**: 配置告警规则、告警通知和告警抑制
8. **集成设置**: 配置CI/CD集成、监控平台集成和安全平台集成
9. **高级功能**: 配置自动修复、多集群管理和机器学习增强
10. **启动验证**: 启动Kubernetes验证器并验证配置
