# 值对象 - 诊断与规划表

## 识别值对象

### 候选对象

```
□ 金额（Money）- 关心值，不关心"哪个"金额 ✓ 值对象
□ 地址（Address）- 两个相同地址就是一样的 ✓ 值对象
□ 用户ID（User）- 每个用户唯一 ✗ Entity
□ 颜色（Color）- 只关心值 ✓ 值对象
□ 订单（Order）- 有唯一订单号 ✗ Entity
□ 邮箱（Email）- 值相等就相等 ✓ 值对象
```

## 实现检查表

```
□ 不可变（final 字段）
□ 自我验证（构造时检查）
□ 无身份标识
□ 值相等（equals 基于属性）
□ 方法无副作用（返回新对象）
```

值对象简化设计，提高类型安全。
