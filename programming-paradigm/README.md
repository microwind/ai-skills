# Programming Paradigm

## 概述

编程范式技能，涵盖面向对象、函数式、响应式等不同编程范式的理论和实践。编程范式是编程思想的结晶，理解它们将帮助你成为更全面的开发者。

## 目标

- 介绍主流编程范式的核心理念
- 分析不同范式的特点和差异
- 提供多语言的范式实现示例
- 指导在不同场景下选择合适的范式

## 适用场景

- 编程范式学习
- 架构设计决策
- 代码风格统一
- 技术栈选型

## 主要内容

### 主要范式
- [面向对象编程 (OOP)](./object-oriented-programming/) - 基于对象和类的编程
- [函数式编程 (FP)](./functional-programming/) - 基于函数和不可变性
- [过程式编程](./procedural-programming/) - 基于过程和函数
- [面向切面编程 (AOP)](./aspect-oriented-programming/) - 横切关注点分离

### 特定范式
- [响应式编程](./reactive-programming/) - 基于事件流和数据流
- [并发编程](./concurrent-programming/) - 多线程和并发处理
- [逻辑编程](./logic-programming/) - 基于逻辑推理
- [数据流编程](./dataflow-programming/) - 基于数据流动

### 新兴范式
- [面向协议编程](./protocol-oriented-programming/) - 基于协议的设计
- [面向数据编程](./data-oriented-programming/) - 优化数据布局
- [面向特征编程](./feature-oriented-programming/) - 特征驱动的开发
- [面向代理编程](./agent-oriented-programming/) - 基于智能代理

### 范式比较
- [OOP vs FP](./oop-vs-fp/) - 面向对象与函数式编程对比
- [范式混合](./paradigm-hybrid/) - 多范式编程实践
- [范式选择指南](./paradigm-selection/) - 如何选择合适的范式
- [语言特性对比](./language-features/) - 不同语言对范式的支持

## 学习路径

1. **基础理解** - 学习主要范式的基本概念
2. **实践对比** - 通过代码示例理解范式的差异
3. **深入掌握** - 选择一两个范式深入学习
4. **混合应用** - 学习多范式编程的实践
