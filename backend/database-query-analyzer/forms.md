## 使用说明

1. **配置顺序**: 按照查询分析基础配置 → 数据库连接配置 → 查询捕获配置 → 查询分析配置 → 数据存储配置 → 报告配置 → 告警配置 → 集成配置 → 高级配置的顺序进行
2. **基础设置**: 配置分析类型、名称、描述和模式
3. **数据库设置**: 配置关系数据库、NoSQL数据库和连接参数
4. **捕获设置**: 配置捕获方式、捕获范围和过滤规则
5. **分析设置**: 配置性能分析、查询优化分析和模式分析
6. **存储设置**: 配置存储策略、数据保留和清理策略
7. **报告设置**: 配置报告类型、生成方式和分发配置
8. **告警设置**: 配置告警规则、通知方式和通知配置
9. **集成设置**: 配置监控系统集成、日志系统集成
10. **高级功能**: 配置机器学习分析、分布式分析和实时分析
11. **启动分析**: 启动数据库查询分析器并验证配置
