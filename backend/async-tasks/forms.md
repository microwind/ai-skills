## 使用说明

1. **配置顺序**: 按照任务基础配置 → 执行配置 → 队列配置 → 调度配置 → 监控配置 → 错误处理 → 通知配置 → 安全配置 → 高级配置的顺序进行
2. **基础设置**: 配置任务类型、名称、描述和优先级
3. **执行配置**: 设置并发控制、超时配置和重试机制
4. **队列设置**: 配置队列类型、参数和消息格式
5. **调度配置**: 设置定时调度和调度策略
6. **监控配置**: 配置性能监控和状态监控
7. **错误处理**: 配置错误策略和日志记录
8. **通知设置**: 配置通知方式和通知条件
9. **安全配置**: 设置访问控制和数据安全
10. **高级功能**: 配置任务链和分布式模式
11. **执行任务**: 启动异步任务并监控执行状态
